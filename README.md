# Task Management System

## Overview

Task Management System is a Python-based application designed to organize and manage daily tasks efficiently. Users can create, update, delete, and monitor tasks in a structured manner.

## Features

* Add tasks
* Update tasks
* Delete tasks
* Mark tasks as completed
* View pending tasks

## Technologies Used

* Python
* Lists and Dictionaries
* Object-Oriented Programming

## How to Run

```bash
python main.py
```

## Learning Outcomes

* CRUD Operations
* Python Programming
* Data Structures
* Problem Solving
